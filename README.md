# React + TypeScript + Vite

## How To Install

- Clone this repo

```
  $ git clone https://github.com/elyasprba/cyber-edu-inkor.git
```

- Go to folder repo

```
  $ cd cyber-edu-inkor
```

- Install module

```
  $ npm install
```

- To start the project

```
  $ npm run start
```
